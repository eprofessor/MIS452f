Setting Up a Raspberry Pi




100 Points
Learning Objectives

These instructions will show you how to download and create the operating system for a Raspberry Pi single-board computer; and then start up the device.
Later you will configure and add various software applications to the Raspberry Pi.

After completing this assignment you will be able to:

  1. Obtain latest Raspbian operating system
  2. Create a bootable micro-SD card containing the operating system
  3. Boot the Raspberry Pi
  4. Connect to the WSU Guest wireless network
  5. Use a web browser and other software on the Raspberry Pi
  6. Install an application on the Raspberry Pi to take screen shots
  7. Do other marvelous things.....

Exercise 1-Etcher
  1. Etcher can be found here: https://www.balena.io/etcher/
  2. Download etcher and make sure you save it to your desktop for easy access.
  3. Downloading the newest Raspbian Operating System (OS).
  4. To download the Raspbian OS go here: https://www.raspberrypi.org/downloads/raspbian/
  5. Make sure to download the version that says with desktop. (have screenshot)
  6. Save the ZIP file to your desktop to make the next steps easier.
  7. Using Etcher to flash your SD Card
Exercise 2- SD card
  1. You will need to take your SD card and put it in your SD card slot which is located on the right side of your laptop. You should feel and hear a click when the SD card is properly seated. It should be flush with the side of your laptop.
  2. If you do not have a SD card slot see Paulson or a TA to get a SD card reader.
  3. Open Etcher and make sure you see that it recognizes that you have an SD card in. (screenshot)
  4. If you do not see it you will need to restart your laptop.
  5. Take the ZIP file that your downloaded earlier and Drag and Drop it into etcher where it says select image.
  6. Then hit Flash (it will take a few minutes).
Exercise 3- Raspberry PI
  1. While that is flashing you will need to locate a keyboard and mouse and plug them into the Pi at your station (We may not have keyboards and mice for everyone so please share).
  2. Once the flash is complete, remove the SD card and put it into the Pi. (gif)
  3. Plug your Keyboard and mouse into the Pi and hit the power button, which should be on top or around the monitor in front of you. You should see lights on the Pi.
  4. Press the button with the purple dot on it.
  5. Follow the on-screen instructions to set your country, language and Time-zone. Check the boxes that say Use English language and Use US keyboard.
  6. Make a password and write it down. If you forget it you will need to start over.
  7. Check “This screen shows a black border around the desktop” if you have a black border.
  8. CONNECT TO WSU GUEST ONLY!!! If prompted make an account and log in.
  9. When asked to update, click next. The update will begin, this make take a few minutes. If asked to restart, do so.
  10. In the upper right corner of the desktop hover over the Wi-Fi signal to determine the IP address.
  11. Write this IP down in the same place as your password. The IP should look something like 10.20.xx.xx
  12. In the upper left corner, you will see the raspberry icon. Click the raspberry icon, then preferences, and Raspberry Pi Configuration.
  13. Click the interfaces tab and Enable SSH and VNC.
Exercise 4- Installing Scrot
  1. In the upper left corner you will see a black icon with >_ written on it. It is called a Terminal. Open the Terminal.
  2. Once the Terminal is open type “sudo apt-get install scrot”.
  3. Once that is complete, open up the browser and navigate to the onedrive website where you will sign into your StarID and password.
  4. Then reopen the terminal if you closed it, type in “scrot”. Make sure that the terminal does not cover your entire screen and that onedrive can be seen in the background.


  1. Close the Terminal and open up File Manager, it should be the icon to the left of the terminal and look like a file .
  2. In your file manager you should a file named something like 2019-mm-dd.


  1. Drag and drop that file into your onedrive.
  2. Once that is completed you can close all windows.
  3. Click the raspberry icon in the upper left and click shutdown.
  4. When the monitor switches back to your lap top desktop you can power off the Pi.
  5. When you no longer see the red light you can safely remove you SD card.
  6. DO NOT REMOVE THE SD CARD WITH THE POWER ON! You can corrupt your SD card and lose everything.
  7. Find the screenshot that you took and upload it to the proper D2L submission folder.
