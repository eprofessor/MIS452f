Exercise 1-Install RPi-Monitor

ref: RPi-Monitor Installation

  1. These steps are performed on the Raspberry Pi.
  2. Open a Terminal window and enter the following commands to ensure your Raspberry Pi is up to date
    sudo apt-get update
    sudo apt-get full-upgrade
  3. In the Terminal window enter the following commands to trust the RPi-Monitor repository:
    sudo apt-get install dirmngr
    sudo apt-key adv --recv-keys --keyserver keyserver.ubuntu.com 2C0D3C0F
  4. In the Terminal window enter the following command to add RPi-Monitor to your repository list:
    sudo wget http://goo.gl/vewCLL -O /etc/apt/sources.list.d/rpimonitor.list
  5. In the Terminal window enter the following commands to install RPi-Monitor:
    sudo apt-get update
    sudo apt-get install rpimonitor
  6. After installation, in the Terminal window enter the following command to update packages information:
    sudo /etc/init.d/rpimonitor update
  7. Open the browser and go to: 127.0.0.1:8888 (if it is not already running)
  8. If the installation was correct you will see a webpage similar to this:

    RPi Monitor

  9. If the webpage does not appear, troubleshoot and resolve any issues.
Exercise 2-Update RPi-Monitor
  1. These steps are performed on your Raspberry Pi.
  2. Open the browser and go to: 127.0.0.1:8888 (if it is not already loaded)
  3. Check the version section, you may need to execute the following command to update the RPiMonitor packages:
    sudo /etc/init.d/rpimonitor update
  4. Examine the information in the different sections of RPi-Monitor.
